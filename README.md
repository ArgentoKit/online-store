## Online Store (Learning Project)<br/>
A fullstack learning project - an online store built to practice backend development.<br/>

### Tech Stack
**Architecture**: Turborepo (monorepo)<br/>
**Backend**: NestJS, Prisma, PostgreSQL<br/>
**Frontend**: Next.js, TailwindCSS, shadcn/ui

**Project Goals**<br/>
Deepening backend skills: NestJS module architecture, working with Prisma (schemas, migrations, table relations), and designing REST APIs for typical e-commerce functionality (product catalog, cart, orders, etc). Also practicing monorepo management with Turborepo.
